# 144. 二叉树的前序遍历

[leecode原题](https://leetcode.cn/problems/binary-tree-preorder-traversal/)

## 题目
给你二叉树的根节点 `root` ，返回它节点值的 `前序` 遍历。

## 示例

### 示例 1:
![](images/inorder_1.jpg)
```text
输入：root = [1,null,2,3]
输出：[1,2,3]
```

### 示例 2:

```text
输入：root = []
输出：[]
```

### 示例 3:

```text
输入：root = [1]
输出：[1]
```

### 示例 4:
![](images/inorder_5.jpg)
```text
输入：root = [1,2]
输出：[1,2]
```

### 示例 5:
![](images/inorder_4.jpg)
```text
输入：root = [1,null,2]
输出：[1,2]
```

## 提示：
- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`

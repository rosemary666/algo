# 77. 组合

[leecode原题](https://leetcode.cn/problems/combinations/)

## 题目
给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k`个数的组合。

你可以按 **任何顺序** 返回答案。

## 示例

### 示例 1:

```text
输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

### 示例 2:

```text
输入：n = 1, k = 1
输出：[[1]]
```

## 提示：
- 1 <= n <= 20
- 1 <= k <= n
